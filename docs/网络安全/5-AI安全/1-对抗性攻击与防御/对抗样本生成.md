# 对抗样本生成

> 对抗样本（Adversarial Example）是指在原始输入上施加人类难以察觉的微小扰动，使机器学习模型产生错误输出的精心构造的输入。这种现象揭示了深度神经网络在输入空间中的脆弱性——人眼看到的是一只熊猫，模型却可能将其分类为长臂猿。

## 基本原理

对抗样本的核心思想是利用模型决策边界的局部线性特性。对于一个分类模型 $f(x)$，攻击者寻找一个微小扰动 $\delta$，使得 $\| \delta \|_p < \epsilon$ 且 $f(x + \delta) \ne f(x)$。扰动通常受 $\epsilon$ 约束以保证不可感知性，常用的范数包括 $L_\infty$、$L_2$ 和 $L_1$。

| 范数类型 | 约束含义 | 典型应用场景 |
|---------|--------|------------|
| $L_\infty$ | 每个像素的最大变化值 | FGSM、PGD |
| $L_2$ | 扰动的欧几里得距离 | C&W、DeepFool |
| $L_1$ | 扰动像素的总数 | 稀疏攻击、patch攻击 |

## FGSM（Fast Gradient Sign Method）

FGSM 由 Goodfellow 等人于 2014 年提出，是最早且最简单的对抗样本生成方法。它基于模型损失函数相对于输入的梯度，沿梯度上升方向添加扰动：

$$x_{adv} = x + \epsilon \cdot \text{sign}(\nabla_x J(\theta, x, y))$$

其中 $\epsilon$ 控制扰动幅度，$\nabla_x J$ 是损失函数对输入的梯度。FGSM 是**单步攻击**，计算效率极高——只需一次前向和一次反向传播。

```python
import torch
import torch.nn as nn

def fgsm_attack(model, x, y, epsilon=0.03):
    """
    FGSM对抗样本生成
    model: 被攻击模型
    x: 原始输入 (batch of images)
    y: 真实标签
    epsilon: 扰动强度
    """
    x_adv = x.clone().detach().requires_grad_(True)
    output = model(x_adv)
    loss = nn.CrossEntropyLoss()(output, y)
    model.zero_grad()
    loss.backward()
    
    # 沿梯度方向添加扰动
    gradient_sign = x_adv.grad.sign()
    x_adv = x_adv + epsilon * gradient_sign
    x_adv = torch.clamp(x_adv, 0, 1)  # 保持在有效范围内
    
    return x_adv.detach()
```

FGSM 生成的对抗样本在 $\epsilon$ 较大时容易出现明显的视觉失真。实际应用中 $\epsilon$ 通常取 0.01~0.1（图像像素值归一化到 [0,1]）。

## PGD（Projected Gradient Descent）

PGD 是 FGSM 的迭代改进版本，由 Madry 等人提出。它在 $\epsilon$ 球内执行多次小步梯度上升，每次迭代后将扰动投影回约束范围：

$$x^{t+1}_{adv} = \Pi_{B_\epsilon(x)} \left( x^t_{adv} + \alpha \cdot \text{sign}(\nabla_x J(\theta, x^t_{adv}, y)) \right)$$

其中 $\Pi$ 是投影操作，$\alpha$ 是步长（通常取 $\epsilon / 4$ 或 $\epsilon / 10$），迭代次数 $T$ 通常为 7~40 步。

| 特性 | FGSM | PGD |
|-----|------|-----|
| 迭代次数 | 1 | 7~40 |
| 攻击强度 | 较弱 | 强（最强一阶攻击） |
| 计算开销 | 低 | 中等 |
| 黑盒迁移性 | 较好 | 一般 |
| 对抗训练防御 | 易被突破 | 标准评估基准 |

```python
def pgd_attack(model, x, y, epsilon=0.03, alpha=0.01, num_iter=40):
    """PGD对抗样本生成"""
    x_adv = x.clone().detach() + torch.randn_like(x) * 0.001
    x_adv = torch.clamp(x_adv, 0, 1)
    
    for i in range(num_iter):
        x_adv.requires_grad_(True)
        output = model(x_adv)
        loss = nn.CrossEntropyLoss()(output, y)
        model.zero_grad()
        loss.backward()
        
        # 梯度上升
        x_adv = x_adv + alpha * x_adv.grad.sign()
        # 投影到epsilon球
        perturbation = torch.clamp(x_adv - x, -epsilon, epsilon)
        x_adv = torch.clamp(x + perturbation, 0, 1).detach()
    
    return x_adv
```

## C&W 攻击（Carlini-Wagner Attack）

C&W 攻击是优化驱动的攻击方法，通过求解以下约束优化问题来寻找最小扰动：

$$\min_{\delta} \| \delta \|_p + c \cdot f(x + \delta) \quad \text{s.t.} \quad x + \delta \in [0, 1]^n$$

其中损失函数 $f(x')$ 定义为 $\max(\max_{i \ne t} Z(x')_i - Z(x')_t, -\kappa)$，$Z(x)$ 是 logits 输出，$t$ 是目标类别，$\kappa$ 控制置信度。C&W 支持三种距离度量：

- **L2 攻击**：使用 Adam 优化器，最常用版本
- **L0 攻击**：迭代式的像素选择，逐步移除不重要像素
- **L∞ 攻击**：使用缩放变量，约束每个像素的变化

C&W 攻击**成功打破了防御蒸馏**，证明了梯度掩蔽类防御的局限性。其缺点是生成速度慢（需要数百次迭代优化）。

## DeepFool

DeepFool 通过迭代线性逼近决策边界来寻找最小扰动。在每次迭代中，它将模型在当前点的决策边界近似为超平面，然后计算到该超平面的最小距离：

```python
def deepfool_attack(model, x, num_classes=10, overshoot=0.02, max_iter=50):
    """DeepFool: 寻找最小对抗扰动"""

    f_x = model(x_adv)
    _, pred = torch.max(f_x, 1)

        _, pred = torch.max(f_x, 1)

            break  # 已经成功欺骗

```

## 迁移性与物理世界攻击

### 对抗样本的可迁移性

对抗样本在不同模型之间具有可迁移性——在一个模型上生成的对抗样本有很大概率能够欺骗另一个模型（即使架构不同）。这是**黑盒攻击**的重要理论基础：

| 迁移因素 | 对迁移率的影响 |
|---------|--------------|
| 源模型与目标模型架构相似度 | 越高迁移率越高 |
| 对抗扰动大小 | 扰动越大迁移率越高 |
| 训练数据的重叠度 | 重叠越多迁移率越高 |
| 攻击方法（迭代 vs 单步） | FGSM迁移性优于PGD |

### 物理世界对抗攻击

数字域的对抗样本在物理世界中面临变换鲁棒性挑战（视角变化、光照变化、几何变换）。物理世界攻击的典型方法包括：

1. **对抗性贴片（Adversarial Patch）**：打印贴片并粘贴到目标物体上，使模型将其识别为其他类别。例如在 Stop 标志上贴对抗性贴片使模型识别为限速标志。

2. **3D 打印对抗物体**：使用 3D 打印技术制造具有对抗纹理的立体物体，验证多视角下的攻击效果。代表性工作包括对 3D 打印的对抗性 turtle 进行分类欺骗。

3. **鲁棒性增强**：通过 EOT（Expectation Over Transformation）框架，在对抗样本生成过程中模拟各种物理变换，提高物理世界攻击成功率。
