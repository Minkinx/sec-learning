# 防御蒸馏与对抗训练

> 对抗样本防御是AI安全的核心研究方向之一，主要分为主动防御（对抗训练）和被动防御（输入变换、梯度掩蔽）两大类。本章深入分析三类代表性防御方法的技术原理、实现细节与已知局限。

## 对抗训练（Adversarial Training）

对抗训练是目前最有效的主流防御方法，其核心思想是将对抗样本纳入训练过程，使模型学会抵抗扰动。

### PGD对抗训练（PGD-AT）

PGD-AT 由 Madry 等人提出，使用 PGD 生成的对抗样本进行训练。其训练目标是最小化对抗样本上的损失：

$$\min_\theta \mathbb{E}_{(x,y) \sim \mathcal{D}} \left[ \max_{\delta \in \mathcal{S}} L(f_\theta(x + \delta), y) \right]$$

内层最大化是寻找最具攻击性的扰动，外层最小化是优化模型参数。

```python
def pgd_adversarial_training(model, dataloader, optimizer, epsilon=0.03, alpha=0.01, num_iter=7):
    """PGD对抗训练主循环"""
    model.train()
    total_loss = 0
    
    for x_batch, y_batch in dataloader:
        x_batch, y_batch = x_batch.cuda(), y_batch.cuda()
        
        # Step 1: 生成PGD对抗样本
        x_adv = pgd_attack(model, x_batch, y_batch, epsilon, alpha, num_iter)
        
        # Step 2: 在对抗样本上训练
        optimizer.zero_grad()
        output = model(x_adv)
        loss = nn.CrossEntropyLoss()(output, y_batch)
        loss.backward()
        optimizer.step()
        
        total_loss += loss.item()
    
    return total_loss / len(dataloader)
```

### TRADES

TRADES（TRadeoff-inspired Adversarial Defense via Surrogate-loss）由张鸣宇等人提出，显式优化了自然精度与鲁棒性之间的权衡：

$$\min_\theta \mathbb{E} \left[ L(f_\theta(x), y) + \beta \cdot \max_{\delta} L(f_\theta(x), f_\theta(x + \delta)) \right]$$

损失函数包含两项：自然损失（保持分类准确率）和边界损失（最小化自然样本与对抗样本的 logits 差异），$\beta$ 控制权衡。

| 特性 | PGD-AT | TRADES |
|------|--------|--------|
| 训练目标 | 最小化对抗损失 | 自然损失 + 正则化项 |
| 自然精度 | 下降较多 | 下降较少 |
| 鲁棒精度 | 较高 | 最高（多个基准中的SOTA） |
| 计算开销 | 高（需要PGD迭代） | 更高（双重优化） |
| 超参数 | epsilon, alpha, iter | epsilon, alpha, iter, beta |

### 对抗训练的代价

对抗训练的主要挑战在于计算开销。在 ImageNet 上使用 PGD-10（10步PGD）训练，训练时间约为普通训练时间的 **10~20 倍**。加速方法包括：

- **免费对抗训练（FreeAT）**：在反向传播中同时更新权重和生成对抗样本
- **YOPO**：只对模型的第一层进行多次PGD迭代，复用后面的梯度
- **Fast AT**：使用 FGSM + 随机初始化 + 提前停止

## 防御蒸馏（Defensive Distillation）

防御蒸馏由 Papernot 等人提出，其核心思想是通过蒸馏平滑决策边界，降低模型对小扰动的敏感性。

### 实现流程

防御蒸馏是一个两阶段过程：

**第一阶段（训练教师模型）**：使用高温 softmax 训练教师网络：

$$\text{softmax}(z, T) = \frac{\exp(z_i / T)}{\sum_j \exp(z_j / T)}$$

其中 $T > 1$ 是温度参数。高温使概率分布更加平滑，捕获类别间的细粒度关系。

**第二阶段（训练学生模型）**：使用教师模型在温度 $T$ 下的 soft 标签训练学生网络，推理时温度降回 $T=1$。

```python
def defensive_distillation(teacher_model, student_model, dataloader, temperature=20):
    """防御蒸馏两阶段训练"""
    # Phase 1: 高温训练教师模型
    teacher_model.train()
    for x, y in dataloader:
        output = teacher_model(x, temperature=temperature)
        loss = nn.CrossEntropyLoss()(output, y)
        # ... 优化步骤

    teacher_model.eval()
    student_model.train()
```

### 局限性

Carlini-Wagner 攻击（C&W L2 攻击）成功打破了防御蒸馏。原因是蒸馏后的模型虽然梯度幅度变小，但梯度方向仍然可用，C&W 使用基于优化的方法绕过了梯度掩蔽。防御蒸馏在大模型场景下仍然被用作一种知识迁移技术，但其安全防护效果已被证明非常有限。

## 特征压缩（Feature Squeezing）

特征压缩通过降低输入空间的复杂度来消除对抗性扰动。核心假设是：对抗扰动在特定特征空间中表现为异常，压缩操作可以滤除这些异常。

| 压缩方法 | 原理 | 对精度影响 |
|---------|------|-----------|
| 位深度缩减 | 将24位色彩降至更少位数（如1位或4位） | 较大 |
| 中值滤波 | 使用中值滤波器平滑图像 | 较小 |
| JPEG压缩 | 利用有损压缩消除高频扰动 | 很小 |
| 空间平滑 | 高斯模糊、均值滤波 | 中等 |

```python
def feature_squeezing_defense(inputs, bit_depth=4):
    """位深度缩减特征压缩"""
    # 将像素值量化到有限的bin中
    max_val = 2 ** bit_depth - 1
    inputs = torch.round(inputs * max_val) / max_val
    return inputs
```

**检测方法**：将压缩前后的模型输出进行比较，差异过大则判定为对抗样本。

## 梯度掩蔽（Gradient Masking）

梯度掩蔽试图通过阻断攻击者可用的梯度信息来防御基于梯度的攻击。

| 掩蔽方法 | 具体技术 | 可绕过性 |
|---------|---------|---------|
| 梯度粉碎 | 使用不可微操作（如argmax参与forward） | 易绕过（使用BPDA） |
| 梯度混淆 | 在梯度计算路径中加入随机噪声 | 可通过期望梯度绕过 |
| 梯度抑制 | 使用分段线性函数使梯度在局部为零 | 可通过BPDA绕过 |

**BPDA（Backward Pass Differentiable Approximation）**：使用可微的近似函数替代原始函数进行反向传播，从而绕过梯度掩蔽：


## 认证鲁棒性（Certified Robustness）

认证鲁棒性旨在为模型提供**形式化保证**——在给定半径 $\epsilon$ 范围内，模型对任意扰动的分类结果不变。

### 随机平滑（Randomized Smoothing）

随机平滑是目前最实用的认证方法，可以对任意模型提供 $L_2$ 范数下的认证鲁棒性：

1. **构造平滑分类器**：$g(x) = \arg\max_c \Pr(f(x + \mathcal{N}(0, \sigma^2 I)) = c)$
2. **认证半径**：$R = \frac{\sigma}{2} \left( \Phi^{-1}(p_A) - \Phi^{-1}(p_B) \right)$

```python
def randomized_smoothing_predict(model, x, sigma=0.25, num_samples=1000):
    """随机平滑预测"""
    predictions = []
    for _ in range(num_samples):
        noise = torch.randn_like(x) * sigma
        pred = model(x + noise).argmax().item()
        predictions.append(pred)
    
    # 统计每个类别的出现次数
    counts = np.bincount(predictions)
    top_class = counts.argmax()
    # 下界置信度（使用Clopper-Pearson区间）
    p_lower = beta.ppf(0.001, counts[top_class] + 1, num_samples - counts[top_class])
    
    return top_class, p_lower
```

### IBP 与 CROWN

**IBP（Interval Bound Propagation）**：将输入扰动建模为区间，通过网络传播区间的上下界，得到输出边界的保证。**CROWN**：使用线性松弛方法计算输出边界。这些方法主要用于训练认证鲁棒的模型。

| 认证方法 | 范数 | 可扩展性 | 认证半径 |
|---------|------|---------|---------|
| 随机平滑 | L2 | 高（可扩展到ImageNet） | 中等 |
| IBP | L∞ | 中（MNIST/CIFAR-10） | 小 |
| CROWN | L∞ | 中 | 中等 |
| CNN-Cert | L∞ | 低（仅小型网络） | 较大 |
