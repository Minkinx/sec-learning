# 网络安全知识指南 · 设计文档

## 概述

制作一份面向有一定经验的安全从业者的网络安全知识指南，帮助系统梳理知识体系。延续与数据安全相同站点风格和架构（VitePress、中文多级目录、知识域导航切换）。

## 技术栈

- **站点生成器**: VitePress（与数据安全指南一致）
- **站点结构**: 网络安全与数据安全同站独立知识库，通过顶部导航切换
- **内容格式**: Markdown (.md)
- **部署**: GitHub Pages / Vercel

## 目录结构

```
docs/
├── 网络安全/

│   ├── 1-基础与核心概念/

```

## 内容框架（6大模块）

### 模块 1：基础与核心概念

建立网络安全的基本认知和基础理论底座。

| 子模块 | 文章 |
|--------|------|
| 1-网络基础 | OSI与TCP-IP模型、常见协议安全、网络拓扑与分段 |
| 2-操作系统安全 | Windows安全机制、Linux安全机制、系统加固基线 |
| 3-密码学 | 对称与非对称加密、哈希与数字签名、TLS-SSL协议深度、国密算法体系 |
| 4-身份认证与访问控制 | 认证机制、授权模型RBAC-ABAC、OAuth-SAML-OIDC、特权账号管理PAM |
| 5-协议安全 | DNS安全、HTTP安全、BGP与路由安全 |

### 模块 2：渗透测试与漏洞利用

覆盖从信息收集到漏洞利用的完整渗透测试链路。

| 子模块 | 文章 |
|--------|------|
| 1-信息收集 | OSINT与被动侦察、主动扫描与指纹识别、子域名枚举与GitHub监控 |
| 2-Web安全 | SQL注入、XSS与CSRF、SSRF与RCE、认证绕过与JWT攻击、文件上传与WebShell、API安全与GraphQL注入、业务逻辑漏洞 |
| 3-内网渗透 | 横向移动、权限提升、域渗透与Kerberos攻击 |
| 4-移动安全 | Android逆向、iOS安全 |
| 5-云安全 | 云配置审计、容器逃逸、Kubernetes安全 |
| 6-无线安全 | Wi-Fi攻击（WPA2-WPA3破解、Evil Twin）、蓝牙安全（BlueBorne、BLE）、RFID-NFC安全 |
| 7-漏洞挖掘与利用 | Fuzzing技术（AFL、LibFuzzer）、逆向工程（IDA Pro、Ghidra）、Exploit开发（Shellcode、ROP）、补丁比对与1-day漏洞 |

## 呈现风格

与数据安全部分保持一致：
- 同站独立知识库，顶部导航切换
- 中文目录命名，编号前缀保持排序
- 多级侧边栏导航，每个模块独立展开
- 每篇文章聚焦一个主题，深度适中
