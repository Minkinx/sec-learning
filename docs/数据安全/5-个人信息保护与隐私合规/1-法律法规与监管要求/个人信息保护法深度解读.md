# 个人信息保护法深度解读

## 法律适用范围

| 维度 | 内容 |
|------|------|
| 适用主体 | 自然人（权利主体）与数据处理者（义务主体） |
| 地域范围 | 在中华人民共和国境内处理自然人个人信息的活动 |
| 域外效力 | 第3条：向境内自然人提供产品或服务、分析评估境内自然人行为的情形适用 |

## 核心定义

| 术语 | 定义要点 |
|------|---------|
| 个人信息 | 以电子或其他方式记录的与已识别或可识别自然人有关的各种信息 |
| 敏感个人信息 | 一旦泄露或非法使用易导致人格尊严或人身财产安全受损的信息（生物识别、金融账户、行踪轨迹等） |
| 自动化决策 | 通过计算机程序自动分析、评估个人行为习惯、兴趣爱好或经济、健康、信用状况等活动 |
| 去标识化 | 处理后无法在不借助额外信息的情况下识别特定自然人的过程 |
| 匿名化 | 处理后无法识别特定自然人且不能复原的过程 |

## 七项处理原则（第5-11条）

1. **合法、正当、必要与诚信原则** — 不得通过误导、欺诈、胁迫等方式处理
2. **目的明确原则** — 处理应具有明确、合理的目的，不得超出目的范围
3. **最小必要原则** — 仅处理实现目的的最小范围信息
4. **公开透明原则** — 公开处理规则，明示处理目的、方式、范围
5. **质量保证原则** — 确保个人信息准确、完整并及时更新
6. **安全保障原则** — 采取必要措施保障个人信息安全
7. **责任原则** — 处理者对其处理活动负责并采取必要措施证明合规

## 六项合法性基础（第13条）

| 序号 | 合法性基础 | 说明 |
|------|-----------|------|
| 1 | 取得个人同意 | 最常用的基础，需单独同意处理敏感信息 |
| 2 | 订立或履行合同所必需 | 合同必要范围内的处理 |
| 3 | 履行法定职责或义务 | 法律法规明确要求的处理 |
| 4 | 应对突发公共卫生事件 | 紧急情况下的必要处理 |
| 5 | 公共利益实施 | 新闻报道、舆论监督等 |
| 6 | 法律法规规定的其他情形 | 兜底条款 |

## 域外适用要点（第3条）

- 在境外处理境内自然人个人信息的活动
- 目的包括：向境内自然人提供产品或者服务；分析、评估境内自然人的行为
- 境外处理者应在境内设立专门机构或指定代表
- 适用本法规定的全部义务，包括告知同意、安全评估等

## 处罚体系

- **一般违法**：警告、没收违法所得、处一百万元以下罚款
- **情节严重**：处五千万元以下或上一年度营业额百分之五以下罚款
- **直接责任人**：一万元以上十万元以下罚款，可禁业
- **诚信记录**：记入信用档案并公示
- **民事赔偿**：过错推定原则，举证责任倒置
