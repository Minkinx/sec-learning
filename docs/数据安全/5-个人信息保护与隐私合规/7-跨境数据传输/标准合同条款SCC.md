# 标准合同条款SCC

## 中国SCC概述

《个人信息出境标准合同办法》（国家网信办2023年2月发布）规定的标准合同条款，为个人信息处理者向境外提供个人信息的法定合规途径之一。

## 中国SCC vs 欧盟SCC对比

| 对比维度 | 中国SCC（2023版） | 欧盟SCC（2021版） |
|---------|-----------------|-----------------|
| 法律依据 | 《个人信息出境标准合同办法》 | 欧盟委员会执行决定2021/914 |
| 合同结构 | 单一模式（个人信息处理者→境外接收方） | 模块化设计（C2C/C2P/P2P/P2C） |
| 强制性 | 合同范本为强制标准，不得修改核心条款 | 可对标准条款进行补充，不得抵触 |
| 备案要求 | 生效后10个工作日内向省级网信办备案 | 无需备案，但应监管要求提供 |
| 监管机构介入 | 网信办可监督合同执行 | 监管机构保留审查权 |
| 适用数据主体 | 中国境内自然人 | 欧盟境内数据主体 |
| 争议解决 | 中国法律管辖、中国法院 | 可约定第三国法律 |
| 个人权利 | 个人作为第三方受益人 | 个人作为第三方受益人 |
| 补充措施 | 无明确规定 | 需进行TIA和补充措施（Schrems II后） |
| 合同修改 | 修改需重新备案 | 补充条款不冲突即可 |

## 中国SCC核心条款

### 基本信息条款

- 合同双方基本信息（名称、地址、联系人）
- 数据出境的目的、范围、方式
- 数据类型和敏感程度
- 传输频率和数量
- 保存期限

### 双方义务条款

#### 个人信息处理者义务

| 义务 | 内容 |
|------|------|
| 告知义务 | 向个人告知境外接收方信息及跨境传输情况 |
| 同意获取 | 获取个人信息主体的单独同意 |
| PIA义务 | 出具个人信息保护影响评估报告 |
| 安全措施 | 采取技术和管理措施保障传输安全 |
| 记录保留 | 保留数据出境记录备查 |

#### 境外接收方义务

| 义务 | 内容 |
|------|------|
| 按约处理 | 严格按照合同约定处理数据 |
| 安全保障 | 采取不低于中国法律要求的安全措施 |
| 事件通知 | 发生安全事件及时通知处理者 |
| 协助义务 | 协助个人信息主体行使权利 |
| 删除/归还 | 合同终止后删除或归还数据 |
| 审计义务 | 允许处理者和监管机构审计 |
| 转委托控制 | 转委托需获得处理者同意 |

### 个人权利条款

- 个人作为合同第三方受益人有权直接主张权利
- 个人有权向境内法院提起诉讼
- 个人有权向网信部门投诉举报
- 因合同履行导致的损失，个人有权索赔

## 备案材料与流程

| 材料 | 要求 |
|------|------|
| 标准合同 | 双方签署的PDF扫描件 |
| 数据出境说明 | 数据类型、数量、目的、方式、频率 |
| PIA报告 | 个人信息保护影响评估报告 |
| 自评估报告 | 数据出境风险自评估报告 |
| 其他材料 | 省级网信办要求的补充材料 |

## 补充措施

在Schrems II判决影响下，无论使用中国SCC还是欧盟SCC，建议采取以下补充措施：

- 对境外接收方的法律环境进行尽职调查
- 评估接收方所在国的数据访问风险
- 实施技术补充措施（加密、假名化、分片存储等）
- 建立数据出境合规台账
- 定期进行合规复核和PIA更新
