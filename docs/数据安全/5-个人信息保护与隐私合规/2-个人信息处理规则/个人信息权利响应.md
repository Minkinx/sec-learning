# 个人信息权利响应

## 个人信息主体权利体系

| 权利 | PIPL条款 | GDPR对应条款 | 内容说明 |
|------|---------|-------------|---------|
| 知情权 | 第44-45条 | Art.13-14 | 了解处理规则和处理情况 |
| 访问权（查阅权） | 第45条 | Art.15 | 查阅个人信息的权利 |
| 更正权 | 第46条 | Art.16 | 要求更正不准确的信息 |
| 删除权 | 第47条 | Art.17 | 法定情形下要求删除信息 |
| 限制处理权 | 未明确列明 | Art.18 | 特定条件下限制信息处理 |
| 可携权 | 第45条第3款 | Art.20 | 结构化、通用、机器可读格式获取信息 |
| 反对权 | 第24条 | Art.21 | 拒绝自动化决策和商业营销 |
| 撤回同意权 | 第15条 | Art.7(3) | 随时撤回同意 |
| 继承权 | 第49条 | 无对应条款 | 去世后近亲属按法律定行使权利 |

## 响应时限对比

| 步骤 | PIPL | GDPR |
|------|------|------|
| 权利请求受理确认 | 无明确规定 | 收到请求后立即确认 |
| 处理时限 | 15个工作日内 | 一个月内（可延长两个月） |
| 费用 | 原则上不得收费 | 一般情况下免费，重复请求可合理收费 |
| 拒绝告知 | 需说明理由 | 需说明理由并告知投诉权利 |

## 身份验证机制

### 验证流程

1. 收到请求后首先确认请求人身份
2. 根据信息敏感程度采用分级验证策略
3. 高风险请求（如删除全部数据）需加强验证
4. 委托他人行使权利需查验授权证明

### 验证方式

| 请求类型 | 建议验证方式 |
|---------|-------------|
| 一般请求 | 注册手机验证码 + 邮箱验证 |
| 身份敏感性请求 | 身份证号码 + 人脸识别 |
| 委托代理人 | 授权委托书 + 双方身份证明 |
| 近亲属继承请求 | 亲属关系证明 + 死亡证明 |

## 豁免情形

### PIPL豁免

- 法律法规规定的保密情形
- 与国家安全、国防安全直接相关
- 与公共安全、公共卫生、重大公共利益直接相关
- 与刑事侦查、起诉、审判和执行判决直接相关
- 个人信息处理者有证据证明个人存在恶意或滥用权利
- 保护个人生命健康和财产安全所必需
- 其他法律法规规定的情形

### GDPR豁免

- 行使言论和信息自由权
- 履行法律义务或执行公共利益任务
- 公共卫生领域的公共利益
- 档案目的的公共利益、科学或历史研究目的、统计目的
- 法律索赔的建立、行使或辩护

## 请求管理流程

### SOP框架

1. **受理阶段**（0-2工作日）
   - 接收请求并记录（请求人信息、请求类型、请求内容）
   - 身份验证
   - 生成工单号并反馈

2. **处理阶段**（3-12工作日）
   - 分发至相关业务部门
   - 数据定位和检索
   - 方案评估（是否存在豁免情形）

3. **执行与反馈阶段**（13-15工作日）
   - 执行处理措施（提供/更正/删除/限制）
   - 书面反馈处理结果
   - 记录存档

### 自动化工具

- 部署在线权利请求门户
- 集成身份验证系统
- 建立自动化工单流转机制
- 使用隐私管理平台（如OneTrust、TrustArc）实现端到端流程
