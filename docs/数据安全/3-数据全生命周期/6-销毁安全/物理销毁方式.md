# 物理销毁方式

## 概述

物理销毁是最彻底的数据销毁方式，通过破坏存储介质的物理结构确保数据完全不可恢复。对于高敏感性数据、废弃设备处置和合规严格要求场景，物理销毁是不可替代的选择。

## 消磁 (Degaussing)

消磁通过强磁场破坏磁介质的磁畴排列，使数据永久性不可恢复。

### 消磁原理

```
消磁过程:
  1. 介质暴露在强交变磁场中 (>10,000 Gauss for HDD)
  2. 磁场强度逐渐衰减至零
  3. 介质上的磁畴随机排列
  4. 数据信号被完全破坏

磁场强度要求:
  介质类型        最低场强      推荐场强
  软盘             1,500 G      3,000 G
  磁带 (LTO)       4,000 G      8,000 G
  HDD (现代)       8,000 G      12,000 G+
  高矫顽力介质     10,000 G     15,000 G
```

### 消磁后的介质状态

- HDD 消磁后**无法重复使用**（磁头寻道信息丢失）
- 磁带消磁后**可重复使用**（需重新格式化）
- 软盘消磁后**可重复使用**（需重新格式化）
- SSD/闪存**不受消磁影响**（无磁性存储单元）

### 消磁优缺点

| 优点 | 缺点 |
|------|------|
| 处理速度快（秒级） | HDD 无法复用 |
| 一次性处理大量介质 | 对 SSD 无效 |
| 无需接触介质表面 | 消磁机昂贵且沉重 |
| 能耗低 | 消磁机需定期校准 |
| 一次性成功率 >99% | 消磁后无法验证数据 |

### 消磁机类型

| 类型 | 处理能力 | 适用场景 | 价格范围 |
|------|----------|----------|----------|
| 柜式消磁机 | 批量 HDD/磁带 | 数据中心 | $5K-$20K |
| 传送带式消磁机 | 连续处理 | 大规模销毁 | $20K-$50K |
| 手持消磁器 | 单盘操作 | 现场处理 | $500-$2K |

## 粉碎 (Shredding)

粉碎将介质切割成微小碎片，物理上阻止任何数据恢复可能性。

### 不同介质的粉碎要求

| 介质 | 推荐碎片尺寸 | 粉碎机类型 | 处理速度 |
|------|-------------|-----------|----------|
| 纸张 | 2mm × 8mm (P-4) / 1mm × 5mm (P-7) | 碎纸机 | 30-100 pages/min |
| HDD | 碎片 < 20mm 或 < 2mm（高安全） | 工业粉碎机 | 1-5 drives/min |
| SSD | 碎片 < 2mm | 工业粉碎机 | 1-3 drives/min |
| 磁带 | 碎片 < 10mm | 工业粉碎机 | 5-10 tapes/min |
| 光盘 | 碎片 < 5mm | 专业光盘粉碎机 | 10-30 discs/min |

### HDD/SSD 粉碎

```
HDD 粉碎流程:
  1. 移除标签和序列号记录
  2. 放入工业粉碎机进料口
  3. 粉碎至指定标准 (通常 < 2mm)
  4. 验证碎片尺寸
  5. 记录视频监控 (可选)
  6. 混合物回收 (金属/塑料分离)

安全等级:
  Level 1: 碎片 < 50mm (数据不可恢复但物理可识别)
  Level 2: 碎片 < 10mm (物理破坏)
  Level 3: 碎片 < 2mm  (最高规格, 符合 NSA/CSS)
```

### SSD 特殊考虑

| 挑战 | 说明 | 应对措施 |
|------|------|----------|
| 闪存芯片耐冲击 | 标准低强度粉碎可能保留可读芯片 | 多次粉碎 + 更高强度 |
| NAND 芯片小而坚固 | 碎片尺寸需更严格 | < 2mm 甚至研磨至粉末 |
| 数据可能保留在废料中 | 部分芯片未被完全破坏 | 目视验证 + 抽样检测 |
| 芯片不完全暴露 | 板层结构保护芯片 | 预先拆解或提高粉碎强度 |

## 焚烧 (Incineration)

焚烧在高温下将介质完全转化为灰烬。

### 温度要求

| 介质类型 | 最低温度 | 持续时间 | 残留物 |
|----------|----------|----------|--------|
| 纸张 | 800°C | 2-5 秒 | 灰烬 (2-3% 体积) |
| HDD/SSD | 1400°C | 15-30 分钟 | 金属渣 + 灰烬 |
| 磁带 | 1000°C | 5-10 分钟 | 灰烬 + 金属颗粒 |
| 光盘 | 1100°C | 3-5 分钟 | 灰烬 |

### 焚烧合规要求

```
焚烧处理需满足:
  - 环保排放标准 (EPA / 地方环保规定)
  - 操作人员 PPE 要求
  - 焚烧温度监控记录
  - 残余物处理合规
  - 废气处理系统 (洗涤器/过滤器)

适用场景:
  - 最高安全级别
  - 大规模销毁
  - 混合介质销毁
  - 国家安全/国防数据
```

## 压碎/粉碎 (Crushing/Pulverizing)

### 压碎 (Crushing)

```
液压压碎:
  - HDD/SSD 放入液压压碎机
  - 施加 10-50 吨压力
  - 破坏盘片/芯片和外壳
  - 处理速度: 5-10 drives/min

优点: 即时验证（目视可见破坏）
缺点: 碎片仍可能被实验室级手段恢复数据
适合: 合同销毁见证 + Purge 后辅助
```

### 研磨 (Pulverizing)

研磨将介质转化为粉末：

```
研磨粒度要求:
  - Level 1: < 5mm (适用于 Confidential 级)
  - Level 2: < 1mm (适用于 Secret 级)
  - Level 3: < 0.5mm (适用于 Top Secret 级)

适用介质:
  - SSD (研磨至粉末确保 NAND 芯片完全破坏)
  - HDD (研磨盘片至金属粉末)
  - 光学介质 (研磨至基材粉末)
```

## 链式监护 (Chain of Custody)

### 销毁全流程记录

```
1. 介质识别阶段:
   - 扫描介质标签 / 序列号
   - 记录介质类型、数量、来源部门
   - 系统录入资产台账
   
2. 运输阶段:
   - 锁定运输容器（防篡改封条）
   - GPS 追踪车辆位置
   - 运输人员双人同行
   
3. 销毁执行阶段:
   - 视频全程监控
   - 销毁参数记录（温度/压力/时间）
   - 执行人在场签名确认
   
4. 验证阶段:
   - 销毁后抽样检查
   - 出具销毁证书
   - 更新资产台账状态
```

### 链式监护表单模板

```
[资产销毁链式监护表]

1. 资产识别
   介质列表: [详见附件 A]
   来源部门: IT 运维部
   销毁原因: 设备报废 / 数据敏感度升级

2. 移交
   移交人: 张三 (签名)    日期: 2024-05-15 10:00
   接收人: 李四 (签名)    日期: 2024-05-15 10:00

3. 运输
   封条编号: SEAL-001-2024
   目的地: [销毁中心地址]
   开始运输: 2024-05-15 10:30
   到达时间: 2024-05-15 11:00

4. 销毁
   销毁方法: 工业粉碎 (< 2mm)
   执行人: 王五 (签名)    日期: 2024-05-15 11:30
   见证人: 赵六 (签名)    日期: 2024-05-15 11:30

5. 验证
   抽样数量: 5 / 100 (5%)
   验证结果: 全部符合标准
   检查人: 钱七 (签名)    日期: 2024-05-15 12:00
```
