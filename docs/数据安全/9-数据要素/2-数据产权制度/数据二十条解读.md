# 数据"二十条"解读

> 2022年12月19日，中共中央、国务院发布《关于构建数据基础制度更好发挥数据要素作用的意见》（简称"数据二十条"），这是我国首份系统构建数据基础制度的纲领性文件。

## 发布背景

"数据二十条"的出台背景包括：
- 数据已成为第五大生产要素，但产权不清、流通不畅、分配不公问题突出
- 数据安全法和个人信息保护法已确立安全底线，需要正向激励制度促进流通
- 数据价值难以评估、交易缺乏标准、市场信任不足

## 四大制度板块

### 1. 数据产权制度
- 建立**数据资源持有权、数据加工使用权、数据产品经营权"三权分置"**的产权制度
- 探索公共数据、企业数据、个人数据的分类分级确权授权
- 推进非公共数据按市场化方式"共同使用、共享收益"

### 2. 流通交易制度
- 完善数据全流程合规和监管规则体系
- 统筹构建规范高效的数据交易场所
- 培育数据商和第三方专业服务机构
- 构建数据安全合规有序跨境流通机制

### 3. 收益分配制度
- 健全数据要素由市场评价贡献、按贡献决定报酬机制
- 兼顾效率与公平，引导数据收益在价值创造者间合理分配
- 建立健全公共数据资源开放收益合理分享机制

### 4. 安全治理制度
- 落实数据安全法、个人信息保护法的安全底线
- 完善数据分类分级保护制度
- 建立数据交易准入和监管机制

## 核心原则

| 原则 | 含义 | 制度体现 |
|------|------|---------|
| 尊重规律 | 顺应数据发展规律，不宜过度干预 | 三权分置替代传统一物一权 |
| 包容创新 | 为数据流通创新留出空间 | 先试点后立法、沙盒监管 |
| 安全有序 | 守住安全底线，确保国家安全 | 分类分级、安全评估 |
| 分类分级 | 区分公共/企业/个人数据 | 差异化确权和授权机制 |
| 市场主导 | 发挥市场决定性作用 | 数据交易所、数据商生态 |
| 政府引导 | 政府侧重规则制定和监管 | 数据基础制度供给 |

## 五大工作原则

1. **数据产权结构性分置**：打破"所有权"思维定势，聚焦使用权流通
2. **合规用权**：在保护个人信息和商业秘密的前提下促进数据使用
3. **协同治理**：政府、企业、社会多方参与
4. **共享红利**：让数据价值惠及全社会
5. **安全可控**：强化数据安全风险评估和监测

## 政策落地进展

- **2023年**：财政部发布《企业数据资源相关会计处理暂行规定》（入表正式实施）
- **2023年**：国家数据局成立，统筹数据要素市场建设
- **2024年**：数据资产入表首个完整会计年度
- **2024年**：各省市出台数据要素市场化配置改革方案
- **2025年**：数据交易所规范化运营深入推进
