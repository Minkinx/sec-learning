# 基因数据隐私保护

> PIPL将基因数据列为敏感个人信息之首，其保护要求显著高于一般医疗数据，涉及存储、处理、研究、跨境等多维度。

## 基因数据的敏感性

| 数据子类 | 示例 | 敏感维度 |
|---------|------|---------|
| 全基因组测序(WGS) | 30亿碱基对全序列 | 一次测序终身关联，可识别个体及其亲属 |
| 全外显子组测序(WES) | 蛋白编码区序列 | 揭示遗传疾病风险、药物代谢能力 |
| 基因检测报告 | 肿瘤驱动基因、遗传病致病基因 | 涉及疾病倾向、预后信息 |
| 基因芯片数据 | SNP位点数据 | 可用于祖源分析、性状预测 |
| ctDNA数据 | 液体活检循环肿瘤DNA | 实时癌症监控数据 |

## PIPL监管要求

| 要求 | 具体规定 | 落地措施 |
|------|---------|---------|
| 单独同意 | 处理基因数据须取得数据主体的单独同意 | 独立同意书，不得混入一般条款 |
| 告知必要 | 必须说明处理目的、方式、范围、保存期限 | 专门隐私条款+操作前弹窗告知 |
| 影响评估 | 处理前必须进行个人信息保护影响评估 | PIA报告存档备查 |
| 本地存储 | 基因数据须存储在中国境内 | 禁止使用海外基因分析平台 |
| 跨境限制 | 原则上禁止出境，特殊情况须安全评估 | 卫健委+网信办双重审批 |

## 科研研究中的基因数据管理

- **知情同意**：研究用基因数据须取得广泛知情同意，涵盖未来用途
- **去标识化**：科研数据应去除直接标识（姓名、身份证号），保留必要表型信息
- **数据共享**：国际科研合作中的基因数据共享须签订数据共享协议
- **退出机制**：受试者有权撤回同意，已去标识化的数据按伦理委员会决定处理
- **收益分享**：研究成果的商业转化应惠及数据贡献者（社区/群体层面）

## 生物样本库治理

| 治理维度 | 要求 |
|---------|------|
| 样本入库 | 知情同意书+伦理委员会审批 |
| 样本出库 | 研究者申请+伦理审查+数据保护评估 |
| 二次使用 | 超出原始同意范围的须重新获取同意 |
| 样本销毁 | 按伦理协议执行，留存销毁记录 |
| 数据关联 | 样本与基因数据的关联标识须加密存储 |

## 国际数据共享限制

- 中国人类遗传资源管理办法对基因数据的出境有严格审批要求
- 涉及国际合作研究的，须与中方合作单位联合申报
- 境外实体不得在中国境内直接采集基因数据
- 基因数据跨境传输必须通过人类遗传资源管理办公室审批
- 违反规定的，最高可处以1000万元罚款并吊销相关资质

## 技术保护措施

- 基因数据存储采用国密SM4加密
- 访问控制采用ABAC模型（基于属性的访问控制）
- 基因数据的计算尽量采用联邦学习或安全多方计算
- 全基因组数据的导出须双人审批
- 密钥与基因数据分离存储，密钥由专人管理
