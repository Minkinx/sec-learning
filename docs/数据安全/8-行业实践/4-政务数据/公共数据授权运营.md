# 公共数据授权运营

> 公共数据授权运营是各级政府通过市场化机制释放公共数据价值的创新模式，须在安全可控前提下开展。

## 授权运营模式

| 模式 | 描述 | 典型实践 |
|------|------|---------|
| 特许经营模式 | 政府通过公开招标授权特定数据运营商 | 上海数据集团、福建大数据集团 |
| 区域一体化模式 | 省级统一授权、地市分域运营 | 浙江、广东 |
| 行业专项模式 | 按行业领域（交通、医疗、金融）分别授权 | 北京金融数据专区 |
| 平台共建模式 | 政府与运营商共建数据运营平台 | 成都公共数据运营平台 |

## 数据运营商选择流程

1. 政府发布授权运营公告，明确授权数据范围、运营期限、安全要求
2. 意向运营商提交运营方案（含数据产品规划、安全措施、收益分配方案）
3. 数据安全委员会评估运营方案的合规性、安全性、可行性
4. 公开招标或竞争性谈判确定运营商
5. 签署授权运营协议（含安全责任条款、退出机制、审计条款）
6. 运营商获得授权，开展数据产品开发

## 数据产品开发

| 产品类型 | 数据来源 | 脱敏要求 | 定价模式 |
|---------|---------|---------|---------|
| API数据接口 | 实时政务数据 | 字段级脱敏 | 按调用量计费 |
| 数据报告 | 统计数据、分析结果 | 聚合级脱敏 | 按份定价 |
| 数据沙箱 | 原始数据+安全计算环境 | 结果差分隐私 | 按时长计费 |
| 行业标签 | 政务标签数据 | 匿名化 | 按标签数量计费 |
| 联合建模 | 政务数据+企业数据 | 隐私计算 | 收益分成 |

## 定价与收费

- **成本回收型**：仅收取数据处理和运营的直接成本（服务器、带宽、人力）
- **价值导向型**：根据数据产品的市场价值定价（适用于高附加值产品）
- **差异化定价**：公益用途（公共服务、科研）免费或低价；商业用途市场化定价
- **收益上缴**：运营收益按约定比例上缴财政，纳入公共预算管理

## 安全监管机制

| 监管环节 | 措施 |
|---------|------|
| 运营准入 | 运营商须通过数据安全能力认证（等保三级以上） |
| 运营监控 | 实时监控所有数据访问、产品调用、异常行为 |
| 定期审计 | 每半年一次数据安全审计，审计报告公开 |
| 安全评估 | 新数据产品上线前须通过安全评估 |
| 退出清算 | 授权到期后运营商须完成数据销毁并出具销毁证明 |
| 公众监督 | 运营情况定期向公众公开，接受社会监督 |

## 去标识化要求

- 授权运营的数据产品不得包含可识别个人身份的原始信息
- 采用去标识化处理（k-anonymity/l-diversity/差分隐私）确保重识别风险可控
- 公共数据开放与授权运营的数据须区分管理，开放数据不得包含高敏感字段
- 数据产品的对外输出结果须经过脱敏复核

## 公共利益保障

- 授权运营不改变公共数据的公共属性，运营商不获得数据所有权
- 数据产品的定价须接受政府指导和价格监管
- 公益类数据产品（公共卫生、防灾减灾）不得拒绝公共服务使用需求
- 授权运营协议须包含公共利益优先条款，应急状态下政府可无条件调用数据
