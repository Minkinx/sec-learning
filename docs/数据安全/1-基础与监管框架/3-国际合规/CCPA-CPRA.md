# CCPA / CPRA

> California Consumer Privacy Act（CCPA）于2020年1月1日生效。California Privacy Rights Act（CPRA）于2023年1月1日生效，是对CCPA的重大修正与扩展。

## CCPA适用范围

适用以下条件之一且在加州开展业务的营利性实体：
- 年总收入超过2500万美元
- 年度买卖/分享10万条（CPRA改为10万）及以上加州居民个人信息
- 年收入50%以上来自出售个人信息

## 消费者权利（CCPA）

| 权利 | 说明 |
|------|------|
| 知情权 | 了解收集、使用、共享的个人信息类别和目的 |
| 删除权 | 要求企业删除已收集的个人信息（例外：完成交易、安全等） |
| 选择退出权 | 拒绝"出售"个人信息给第三方 |
| 非歧视权 | 行使权利不影响服务质量和价格 |

## CPRA主要变更

- **新增敏感个人信息（SPI）**：精确地理位置、种族、社保号、基因数据等——消费者可限制其使用
- **新增更正权**：允许消费者要求更正不准确的个人信息
- **新增自动化决策知情权**：了解自动化决策的运作方式
- **设立新执法机构**：加州隐私保护局（CPPA），专职执法
- **DPIA要求**：存在高风险处理时须开展数据保护影响评估
- **dark patterns禁止**：禁止通过诱导性设计阻碍用户行使权利
- **数据最小化**：收集目的变更须获得新同意

## 与GDPR对比

| 维度 | CCPA/CPRA | GDPR |
|------|-----------|------|
| 同意 | opt-out模式 | opt-in模式（严格同意） |
| 范围 | 加州消费者 | 欧盟数据主体 |
| 处罚 | 每次违规最高$7,500 | 全球营收4%或€20M |
| 被遗忘权 | 有限删除权 | 更广泛 |
| 数据可携带 | 仅限电子格式 | 明确要求互操作格式 |
