# 关键信息基础设施安全保护条例

> 《关键信息基础设施安全保护条例》于2021年9月1日与《数据安全法》同步施行。

## CII定义与认定

关键信息基础设施（CII）是指公共通信和信息服务、能源、交通、水利、金融、公共服务、电子政务、国防科技工业等重要行业和领域的，以及其他一旦遭到破坏、丧失功能或数据泄露，可能严重危害国家安全、国计民生、公共利益的重要网络设施、信息系统等。

**认定流程**：
1. 行业主管部门制定认定规则
2. 运营者自行申报
3. 主管部门审核确认并通知运营者
4. 定期复审和动态调整

## 安全保护义务

| 义务类别 | 具体要求 |
|---------|---------|
| 组织建设 | 设立专门安全管理机构，明确负责人 |
| 人员管理 | 对关键岗位人员进行安全背景审查 |
| 制度流程 | 建立网络安全保护制度、应急预案 |
| 技术防护 | 落实等级保护三级以上要求 |
| 供应链安全 | 优先采购安全可信的产品和服务 |

## 安全检测评估

- 运营者**每年**至少进行一次安全检测评估
- 检测评估须由具备相应资质的机构进行
- 评估结果报送主管部门

## 产品和服务安全

- CII运营者采购的网络产品和服务可能影响国家安全的，须通过国家安全审查
- 采购时应签订保密协议和安全承诺书
- 优先使用自主可控技术

## 法律责任

- 未履行保护义务：最高罚款100万元
- 情节特别严重：最高罚款5000万元或上年营收5%
- 直接责任人：最高20万元罚款
