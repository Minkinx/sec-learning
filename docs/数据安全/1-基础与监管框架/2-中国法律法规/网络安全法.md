# 网络安全法

> 《中华人民共和国网络安全法》于2017年6月1日正式施行，是我国网络安全领域的基础性法律。

## 适用范围

- 在境内建设、运营、维护和使用网络
- 通过网络开展业务活动
- 关键信息基础设施运营者

## 核心制度

### 等级保护制度（等保2.0的基础）
法律要求按照国家规定的网络安全等级保护要求，对网络系统实行安全保护。

### 关键信息基础设施（CII）保护
对公共通信、能源、交通、金融、电子政务等重要行业和领域实行重点保护。

### 个人信息保护
网络运营者收集、使用个人信息须遵循合法、正当、必要原则，并经用户同意。

### 重要数据本地化
CIIO在境内运营中收集和产生的个人信息和重要数据应当在境内存储。

## 关键义务主体

| 义务主体 | 主要义务 |
|---------|---------|
| 网络运营者 | 履行安全保护义务，落实等级保护制度 |
| CIIO | 设置专门安全机构、定期安全检测评估、数据本地化 |
| 网络产品/服务提供者 | 不得设置恶意程序，发现漏洞及时补救 |

## 安全事件报告

- 发生安全事件后应当立即启动应急预案
- 及时向有关主管部门报告
- 事件处置情况应当予以记录

## 法律责任

| 违规类型 | 罚款上限 |
|---------|---------|
| 未履行安全保护义务 | 100万元 |
| 侵害个人信息 | 违法所得五倍以下或100万元 |
| CIIO违规 | 100万元 |

## 与相关法律的关系

- **网络安全法**是总体框架，侧重网络空间安全
- **数据安全法**在数据安全领域进行细化和扩展
- **个人信息保护法**专门针对个人信息权益保护
- 三部法律共同构成我国"三法合一"的数据治理法律体系
