# 个人信息保护法

> 《中华人民共和国个人信息保护法》（PIPL）于2021年11月1日正式施行，是中国首部专门针对个人信息保护的法律。

## 适用范围

- **属地原则**：在中国境内处理个人信息的活动
- **属人原则**：向境内自然人提供产品/服务、分析评估境内自然人行为的境外处理活动

## 关键定义

| 术语 | 定义 |
|------|------|
| 个人信息 | 以电子或其他方式记录的与已识别或可识别自然人有关的各种信息 |
| 敏感个人信息 | 一旦泄露或非法使用，容易导致人格尊严或人身财产安全受损的信息（生物识别、金融账户、行踪轨迹等） |
| 处理者 | 自主决定处理目的、处理方式的组织、个人 |
| 委托处理 | 处理者委托第三方代表其处理个人信息 |

## 七项处理原则

1. 合法、正当、必要
2. 目的明确
3. 公开透明
4. 准确性
5. 最小必要
6. 安全可控
7. 责任主体清晰

## 法律依据

处理个人信息须具备以下之一：知情同意、履行合同所必需、履行法定职责、应对突发公共卫生事件、为公共利益实施新闻报道、法律/行政法规规定的其他情形。

## 个人权利

- 查阅、复制、更正、补充
- 删除（撤回同意、目的已实现、违反法律等情形）
- 可携带权（转移至指定处理者）
- 撤回同意
- 自动化决策的知情权与拒绝权
- 死者近亲属查阅/复制权

## 法律责任

| 情形 | 罚款 |
|------|------|
| 一般违规 | 最高100万元 |
| 情节严重 | 最高5000万元或上年营收5% |
| 直接责任人 | 最高10万元，并可禁止担任相关职务 |

## 域外适用

PIPL明确具有域外效力。境外处理者须设立境内专门机构或代表。这将与GDPR形成跨境执法的互动与张力。
